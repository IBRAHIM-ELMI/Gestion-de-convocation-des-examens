 \section{Qu\' est ce que SVG}
\begin{frame}
\frametitle{\textcolor{white}{\textbf{Qu'est ce que }}}
\begin{block}{{\textbf{Qu' est ce que SVG}}}
        
SVG(Scalable Vector Graphics) est un langage xml qui permet d’écrire des
graphiques vectoriels 2D. Il a été inventé en 1998 par un groupe de travail (comprenant
Microsoft, Autodesk, Adobe, IBM, Netscape, Xerox, Apple, HP, ILOG...) pour répondre à
un besoin de graphiques légers, dynamiques et interactifs.
En octobre 1998, une première ébauche du langage est sortie. Ainsi, en juin 2000 apparaît
la première version du Viewer Adobe (plugin permettant de visualiser le svg). Adobe a
intégré ce langage dans la plupart de ses éditeurs (comme Illustrator).

        
        \end{block}
\end{frame}

\section{Format SVG}
\begin{frame}
\frametitle{\textcolor{white}{\textbf{ Format SVG }}}
    \begin{block}{\textbf{Structure de base}}
        
\item[$\bullet$] Dans tous documents SVG doit toujours commencer par la déclaration XML, comme
ceci : \\ \\ \pause 
\begin{itemize}
\item[$\sqrt$] <?xml version="1.0" standalone="no"?> \\ \pause  
\end{itemize}

 \item[$\sqrt$] La DTD du SVG s’inclut avec la balise < !DOCTYPE>, comme ceci : \\ \pause 
 \begin{itemize}
 \item[$\sqrt$] <!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd"> \\ \pause 
  \end{itemize}
 
 
 \item[$\sqrt$] La taille de la fenêtre SVG est définie par les attributs width et height de l’element svg : \\ \pause 
 \begin{itemize}
 \item[$\sqrt$] <svg width="15.0cm" height="15.0cm" viewBox="100 100 200 600" zoomAndPan="disable" xmlns="http://www.w3.org/2000/svg" version="1.1"> \\ \pause 
  \end{itemize}

void addPoint(Point p) {
if (empty) {
empty = false;
x_max = p.getX();
x_min = p.getX();
y_max = p.getY();
y_min = p.getY();
} else {
if (x_max<p.getX()) x_max = p.getX();
else if (x_min>p.getX()) x_min = p.getX();
if (y_max<p.getY()) y_max = p.getY();
else if (y_min>p.getY()) y_min = p.getY();
}
points.add( p);
} // fin méthode addPoint

public void add(int x, int y) {
addPoint(new Point (x,y));
} // fin méthode add

} // fin classe Courbe
